# Pengembangan-aplikasi-peminjaman-buku
pengembangan aplikasi peminjaman buku dan pengelolaan perpustakaan sekolah berbasis web.


Aplikasi ini dibuat untuk membantu pengelolaan perpustakaan sekolah berbasis web, 
meliputi proses peminjaman buku, pengembalian buku, serta manajemen data buku dan anggota
agar lebih terstruktur, efektif, dan efisien.

## Tujuan
- Mempermudah proses peminjaman dan pengembalian buku
- Mengelola data buku dan data anggota perpustakaan
- Mengurangi kesalahan pencatatan data secara manual
- Meningkatkan efisiensi administrasi perpustakaan sekolah

## Fitur Utama
- Daftar
- Login
- Pemilihan Menu
- Melakukan CRUD Transaksi
- Melakukan CRUD Data Buku
- Melakukan CRUD Kelola Anggota
- Pencarian untuk Kondisi Tertentu

## Hak Akses Pengguna
- **Admin**
  - Mengelola data buku (CRUD)
  - Mengelola data anggota (CRUD)
  - Mengelola transaksi peminjaman dan pengembalian
  - Melihat laporan peminjaman

- **User**
  - Login ke sistem
  - Melakukan peminjaman buku
  - Melakukan pengembalian buku
  - Melakukan pencarian buku

## Teknologi yang Digunakan
- XAMPP
- VSCODE

## Arsitektur Sistem
Aplikasi ini dikembangkan menggunakan framework Laravel dengan konsep fullstack,
di mana backend dan frontend terintegrasi dalam satu aplikasi.
